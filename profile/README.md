## What You’ll Find Here

This GitHub organization hosts the codebases for Albaidex’s software projects, internal tools, and client applications. We embrace open-source principles where appropriate, allowing visibility into our work while protecting commercial rights.

* **Web Applications** — Frontend and backend projects built with React, Next.js, Node.js, PHP, and more.
* **Internal Tools** — Software that powers our workflows and client services.
* **Open Source** — Public projects that reflect our commitment to transparency and community collaboration.

## How to Get Started

To explore our projects locally:

1. Clone the desired repository from this organization.
2. Install dependencies with `npm install` or `composer install` (depending on the project).
3. Follow each repo’s specific README instructions for running and development.
4. Use GitHub Issues and Pull Requests to collaborate with the team.
